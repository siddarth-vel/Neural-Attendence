# NeuralMark AI — Intelligent Attendance Platform

A smart attendance system that uses computer vision and AI to automate
student attendance tracking in real time — no roll calls, no proxy fraud.

## Features
- Real-time multi-face detection (up to 32 faces simultaneously)
- 6-factor trust score engine (face match, liveness, anti-spoof, engagement)
- Emotion & engagement analytics per session
- Unknown person detection with instant alerts
- Predictive at-risk student flagging (Isolation Forest ML)
- AES-256 encrypted biometric storage
- Auto-generated CSV / PDF attendance reports
- Live session dashboard with detection log

## Tech Stack
Python · OpenCV · FaceNet 128D · MediaPipe · TensorFlow
DeepFace · dlib · Flask · SQLite · AES-256 · Scikit-learn
